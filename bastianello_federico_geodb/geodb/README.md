## Descrizione
programma che elabora un file csv passato da CLI e inserisce il risultato dell'elaborazione all'interno del db.

## requisiti
python 3.10 >=
librerie python non standard (guarda requirementsd.txt)

## Esecuzione
da shell:
python ./geodb.py ../flussi/location.csv
